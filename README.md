app的简要介绍：1.查看实况天气以及未来几天的天气，可以手动选择你需要查看的城市
              2.实况天气用了一个普通的界面布局，未来天气采取的RecyclerView以及适配器来实现左右滑动查看，然后用网络请求方法弄了一个天气api获取的天气数据
              然后运用了handler接收了子线程传过来的信息，还用了一个gson库，再把数据解析出来，填充到界面对应的控件上面去。
